# Features

<h2 id="soc">SOC / Threat Detection</h2>

- FortiSIEM alert monitoring and triage
- IOC analysis and log correlation
- Incident reporting and escalation workflows

<h2 id="automation">Security Automation</h2>

- Python scripts for faster triage and correlation
- Structured outputs for repeatable investigations

<h2 id="smart-contracts">Smart Contract Security</h2>

- Static analysis with Slither + Mythril
- ML-based risk scoring and prioritization (in development)

<h2 id="cloud">AWS CloudTrail Analysis</h2>

- CloudTrail investigation for suspicious activity
- Event reconstruction and timeline building
